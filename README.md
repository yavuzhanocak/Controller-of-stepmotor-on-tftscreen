# controller-of-step-motor-on-tft-screen
pic+proteus+realtime app 

## Elements used in the project
 - PIC 18F877A
 - NEXTİON HMI 2.8 TFT 
 - Two units 28BYJ-48 STEPPER MOTOR
 - Two units ULN2003A
 - ROTARY ENCODER
